Mes Projets

RSA - Datasafe

Implémentation de l'algorithme de chiffrement RSA• Février 2020 - Avril 2020

Projet de fin de licence en groupe concernant l'implémentation de l'algorithme de chiffrement RSA. Nous avons élaboré une application en langage C permettant à l'utilsateur de chiffrer des fichiers ou des messages avec l'algorithme RSA, le tout dans une interface intuitive, ergonomique et épurée. L'implémentation a été faite par nos soins de la génération de clés jusqu'aux différentes fonctionnalités de l'application.

Projet de recherche sur l'intégration de sources dans une base de donnée

Intégration de sources de données pour l'analyse du piratage des logiciels dans le monde• Février 2021 - Avril 2021

Projet de première année de Master à faire en groupe concernant l'intégration de sources de données sur le piratage des logiciels. Nous travaillons avec une équipe de chercheurs en Sciences humaines et sociales. Dans le but d'approfondir leur recherche et de faciliter la manipulation de données pour tous, nous avons conçu un programme en Python permettant à tout utilisateur disposant de l'application d'installer automatiquement et simplement la base de données depuis différentes ressources web de différent format vers un seul et unique format final, sans stockage physique permanent des fichiers de données recueilli grâce à l'élaboration d'un ETL. Ainsi, l'utilsateur peut grâce à une application Python importer une base de données interactive avec le web et générant un format lisible pour tous ainsi qu'une représentation graphique des résultats recueillis notamment par la création automatique de carte mondiale avec des légendes en adéquation à la données traitée.
